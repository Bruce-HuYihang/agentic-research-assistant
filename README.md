# Agentic Research Assistant - Phase 1

## 目录
- `app/config.py` - 全局配置
- `app/agent/state.py` - Agent 状态模型
- `app/tools/search.py` - 百度搜索
- `app/tools/fetch.py` - 网页爬取
- `app/agent/nodes/planner.py` - 规划节点
- `app/agent/nodes/generator.py` - 生成节点
- `app/agent/graph.py` - LangGraph 定义
- `app/__main__.py` - 命令行入口
